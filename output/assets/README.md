# Wiki Visual Assets

This folder contains demonstrative images and diagrams for the boat license wiki.

## Naming Convention

To ensure images are correctly wired into the wiki notes, please use the following filenames for core assets:

### Balisage (Buoyage)
- `bouee-cardinale-nord.png`
- `bouee-cardinale-sud.png`
- `bouee-cardinale-est.png`
- `bouee-cardinale-ouest.png`
- `bouee-laterale-babord.png`
- `bouee-laterale-tribord.png`
- `bouee-danger-isole.png`
- `bouee-eaux-saines.png`
- `bouee-speciales.png`

### Feux & Marques (Lights & Shapes)
- `feu-mouillage.png`
- `feu-peche.png`
- `marque-voilier-moteur.png`

### Questions
- `question-<theme>-<NN>.png` (e.g., `question-balisage-01.png`)

## Implementation Note
Images extracted from course PDFs land in this directory. If an automated extraction produces thousands of fragments, please manually identify and rename the definitive diagrams to match the slugs above.
