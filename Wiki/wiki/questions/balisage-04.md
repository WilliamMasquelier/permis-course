**Q:** Vous naviguez vers le port, votre tirant d'eau est important, droit devant ce feu **à deux éclats verts + 1** :
- A — je viens sur tribord
- **B — je viens sur bâbord** ✓
- C — indifféremment

**A:** B — je viens sur bâbord.

**Why:** Feu vert 2 éclats + 1 = balise de chenal préféré **bâbord** (couleur dominante verte). Sens conventionnel (vers le port) → je dois laisser cette marque sur bâbord → je me dirige sur bâbord pour passer du côté profond (chenal préféré = passage le plus profond). Voir [[../concepts/marques-laterales]].
