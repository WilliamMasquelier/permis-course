# Question
Quel nœud utiliseriez-vous pour former une boucle solide qui ne coulisse pas ?

## Options
A) Le nœud de cabestan
B) Le nœud de chaise
C) Le nœud de taquet
D) Le nœud plat

## Answer
B

## Explanation
Le nœud de chaise permet de faire une boucle (un œil) très solide qui ne se resserre pas sous la tension, tout en restant facile à défaire.
