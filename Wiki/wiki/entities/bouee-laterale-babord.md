**Summary:** Balise rouge à laisser sur bâbord (gauche) quand on navigue dans le sens conventionnel (vers le port).

| Attribut | Valeur |
|----------|--------|
| Couleur | Rouge |
| Forme | Cylindrique (espar ou carotte) |
| Voyant | Carré rouge |
| Numéro | Chiffre **pair** (2, 4, 6…) |
| Feu nocturne | **Rouge**, rythme quelconque |
| Région A | Je la laisse à **gauche** en allant vers le port |
| Région B | Couleurs inversées — voyant = indicateur fiable |
