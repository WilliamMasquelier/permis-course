**Summary:** Les infractions nautiques vont de la contravention de 1ère classe (38 €) à 5ème classe (1500 €) ; certaines comportent des peines d'emprisonnement et/ou amendes pénales lourdes.

## Barème des contraventions

| Classe | Montant maximum |
|--------|----------------|
| 1ère | 38 € |
| 2ème | 150 € |
| 3ème | 450 € |
| 4ème | 750 € |
| 5ème | 1 500 € |
| Récidive 5ème | **3 000 €** |

## Tableau des principales infractions

| Infraction | Classe / Sanction |
|------------|------------------|
| Défaut de titre de navigation | 5ème classe (1 500 €) |
| Pas de marques extérieures | 5ème |
| Défaut de documents, route et signaux | 5ème |
| Défaut d'appareils, instruments, documents | 5ème |
| Conduite sans permis | 5ème |
| Défaut de présentation immédiate | 1ère |
| Vitesse excessive dans la bande des 300 m | **3 750 € + 6 mois d'emprisonnement** |
| Vitesse excessive dans les ports | 3ème |
| Surnombre | 5ème |
| Défaut matériel d'armement / rechange | 5ème |
| Défaut d'extincteur ou de signaux de détresse | 5ème |
| **Non-assistance à personne en détresse** | **3 750 € + 2 ans d'emprisonnement** |
| Défaut d'allumage des feux la nuit + brume | **3 750 € + 3 mois d'emprisonnement** |
| Vente des produits de sa pêche par un plaisancier | **22 500 €** |
| Pêche sous-marine avec foyer lumineux | 5ème |

## Points clés

- **Nolisage** (location d'un navire) : en cas d'accident, le **titulaire du permis** est responsable, pas le loueur (noliseur).
- Défaut d'instruments ou de documents → contravention de **5ème classe** (et non 3ème comme on pourrait le penser).
- Les symboles et abréviations sur les cartes sont répertoriés dans le **livre du SHOM**.
- La pêche sous-marine de loisir **ne peut pas** se pratiquer avec un équipement respiratoire.
- Homme-grenouille : **interdit de détenir simultanément** un appareil respiratoire + un appareil pour la pêche sous-marine à bord.
