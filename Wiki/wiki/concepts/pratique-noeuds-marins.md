# Les nœuds marins

Savoir réaliser les nœuds fondamentaux est obligatoire pour assurer la sécurité du navire à l'amarrage.

## 1. Le nœud de taquet
- Il permet de fixer une amarre sur un taquet (sur le bateau ou sur le ponton).
- Il ne se bloque pas sous la tension.
- **Réalisation :** On fait un tour mort (un tour complet) autour de la base, puis on fait des 8 en croisant par-dessus le taquet, et on termine par une demi-clé (la corde passe sous elle-même) bloquée sur une corne.

## 2. Le nœud de chaise
- Forme une boucle (un œil) fixe qui ne coulisse pas (il ne serre pas l'objet autour duquel il est passé).
- Très solide sous tension, mais reste toujours facile à défaire même après avoir subi de fortes charges.
- *"Le serpent sort du puits, fait le tour de l'arbre et rentre dans le puits."*

## 3. Le nœud de cabestan
- Permet d'amarrer un cordage sur un cylindre, un pieu, ou pour accrocher un pare-battage sur les filières.
- Très facile à régler en hauteur. Il peut toutefois glisser si la tension n'est pas constante. Il est souvent sécurisé par une demi-clé supplémentaire.
