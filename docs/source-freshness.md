# Source Freshness

Last checked: 2026-05-07.

## Official Exam Rules

The French Ministry page for the *permis plaisance* states that, for the exam,
40 questions are asked and 5 errors are admitted. The Ministry reform notice
also states that since 2022 the theory code moved from 30 to 40 questions with
5 tolerated errors.

Primary sources:

- Ministry permit overview: https://www.mer.gouv.fr/le-permis-plaisance-permis-de-conduire-les-bateaux-de-plaisance-moteur
- Ministry 2022 theory exam reform: https://www.mer.gouv.fr/lexamen-theorique-du-permis-plaisance-evolue
- ANFR CRR and maritime radio: https://www.anfr.fr/gerer/radiomaritime/le-crr

## Local Course Assumptions

- Mock exam pass threshold: 35/40.
- Local mock exam uses the repo's current question bank, not an official exam
  feed.
- VHF/CRR material is included because it appears in the source corpus and is
  useful for safety. The coastal permit allows VHF use in French territorial
  waters; CRR remains relevant for broader radio use cases.

## Maintenance Rule

Recheck these sources at least yearly, and immediately before presenting this
course as exam-ready to another learner.
