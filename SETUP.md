# Guide d’installation - Permis Côtier Masterclass

## Ce dont vous avez besoin

1. **Claude Code** - l’application IA (installez-la depuis claude.ai/code)
2. **Un compte Claude** - créez-en un sur claude.ai
3. **Ce plugin** - installez-le en une commande (voir ci-dessous)

## Installation en 3 étapes

### Étape 1 - Installez le plugin

Dans Claude Code, tapez :

    /install permis-course

*(ou ouvrez ce dossier directement dans Claude Code)*

### Étape 2 - Vérifiez l’installation

Tapez :

    /permis-setup

Claude vérifiera que tout est en ordre et vous guidera si quelque chose manque.

### Étape 3 - Commencez le cours

Tapez :

    /permis-tutor

Le tuteur ouvrira la première leçon dans votre navigateur et commencera à vous enseigner.

## Commandes disponibles

| Commande | Description |
|----------|-------------|
| /permis-tutor | Démarrer ou reprendre une session d’enseignement |
| /permis-scenario | Exercice pratique de navigation |
| /permis-exam | Examen blanc (30 questions, 45 min) |
| /permis-render | Mettre à jour les pages HTML |
| /permis-setup | Vérifier l’installation |

## Questions fréquentes

**Le navigateur ne s’ouvre pas automatiquement ?**
Pas de problème - le tuteur vous donnera l’URL à ouvrir manuellement.

**Puis-je reprendre où j’en étais ?**
Oui - votre progression est sauvegardée automatiquement dans Wiki/meta/student-progress.json.

**Comment refaire une session ?**
Demandez au tuteur : "Je voudrais revoir la session 1."
